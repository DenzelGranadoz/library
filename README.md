## Features and Usage
- LocalStorage
- Adding a book
- Deleting a book
- Total number of Books is displayed on the top
- Total number of READ Books is displayed on the top
- Every book have a read and not read toggle button.
- Add book form which contains all the information of the book.
- Book has Author, Book Title and Number of pages displayed.
